# دليل المشروع — موقع رياض الشهري

## ملخص المشروع

موقع شخصي لرياض الشهري، طالب هندسة برمجيات في جامعة أم القرى (18 عامًا). يعرض مقالات باللغتين العربية والإنجليزية، ومشاريع برمجية. الهدف: موقع شخصي احترافي يعكس أسلوبًا تحريريًا تحليليًا ورصينًا.

## التقنيات المستخدمة

- **الإطار:** Next.js 14+ مع App Router
- **اللغة:** TypeScript (strict mode)
- **التنسيق:** Tailwind CSS v4
- **المكونات:** shadcn/ui
- **مدير الحزم:** npm

## فلسفة التصميم

- **التبسيط الراقي:** لا زخرفة مجانية — كل عنصر بصري له وظيفة.
- **الطباعة هي الأساس:** الخط والحجم والمسافة هي أدوات التصميم الأولى.
- **لون التمييز الزيتوني:** `#4a6741` في الوضع الفاتح، `#7a9970` في الداكن. يُستخدم بحذر فقط للعناصر التفاعلية والتأكيدات البصرية.
- **العربية أولًا:** التصميم يبدأ من اليمين إلى اليسار، مع دعم كامل للإنجليزية.

## قواعد العمل

1. **مرجع واحد للتخصيص:** جميع الألوان والخطوط ومعلومات الموقع تُعرَّف في `theme.config.ts` حصرًا. لا ألوان مشفرة داخل المكونات.
2. **لا إيموجي في الكود:** يُمنع استخدام الرموز التعبيرية في أي ملف برمجي أو CSS.
3. **اقترح قبل التعديل الكبير:** عند أي تغيير جوهري في البنية أو التصميم، اعرض الفكرة أولًا وانتظر الموافقة.
4. **لا صفحات أو محتوى تجريبي:** لا تنشئ صفحات أو مكونات مؤقتة دون طلب صريح.
5. **TypeScript صارم:** جميع الأنواع معرَّفة، لا استخدام لـ `any`.

## تفضيلات شخصية

- **أسلوب الكتابة العربية:** رسمي حديث، مثل أسلوب ثماني — دقيق، تحليلي، بلا تكلف.
- **النبرة:** مهنية وأكاديمية، تناسب القارئ المتخصص في التقنية.
- **الخط العربي المفضل:** خطوط ثماني (serif display للعناوين، serif text للمقالات، sans للواجهة).
- **الخط الإنجليزي:** IBM Plex — serif للمقالات، sans للواجهة، mono للكود.
